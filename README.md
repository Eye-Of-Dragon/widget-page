# test at docker:
go to the repository folder then run the commands below:
 - ``cd devops``
 - ``docker-compose build``
 - ``docker-compose up``
then go to `http://localhost:8080/#/`


# test local machine:
## requirement:
  node: 12.0.0
  npm: 6.9.0
execute commands below:
 - ``npm install -D``
 - ``npm run start:dev``
then go to `http://localhost:8080/#/`

# to run test:
execute commands below:
 - ``npm run test``
